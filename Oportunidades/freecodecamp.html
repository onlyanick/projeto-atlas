Programa

O freeCodeCamp é uma organização sem fins lucrativos que tem como objetivo dar a qualquer pessoa a oportunidade de ingressar na área de tecnologia por meio da educação gratuita e acessível em programação, oferecendo um currículo interativo e amplo.

O programa Full-Stack Web Development & Machine Learning é um currículo autodidata que acompanha o aluno do nível iniciante até habilidades avançadas em desenvolvimento e ciência de dados. A organização oferece acesso totalmente gratuito a mais de 3.000 horas de desafios de programação, cursos em texto ou vídeo e certificações.

Cursos disponíveis

  • Certificação de Design Responsivo para a Web

    Este curso ensina os fundamentos de HTML e CSS, incluindo design de layout moderno, acessibilidade e desenvolvimento web responsivo. Você criará projetos práticos e adquirirá as habilidades necessárias para criar páginas da web profissionais e fáceis de usar.

  • Certificação JavaScript

    Este curso ensina conceitos fundamentais de programação JavaScript, como variáveis, funções, objetos, arrays e controle de fluxo. Você também aprenderá a manipular o DOM, lidar com eventos e aplicar técnicas como programação funcional, assíncrona e práticas recomendadas de acessibilidade.

  • Certificação de Bibliotecas de Desenvolvimento Front-End

    Este curso ensina as principais bibliotecas utilizadas por desenvolvedores para criar páginas da web modernas, incluindo React, TypeScript e mais.

  • Certificação em Python

    Este curso ensina os fundamentos da programação em Python, uma das linguagens mais utilizadas no mercado para desenvolvimento web, ciência de dados e automação.

  • Certificação de Bancos de Dados Relacionais

    Este curso ensina os fundamentos dos bancos de dados relacionais, incluindo modelagem de dados e consultas com SQL.

  • Certificação da Microsoft em C# (disponível apenas em inglês)

    Este curso oferece uma introdução abrangente à programação em C#, abordando seus conceitos fundamentais, sintaxe e aplicação prática no desenvolvimento de software.
